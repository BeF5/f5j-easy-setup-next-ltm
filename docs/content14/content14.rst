================================================
Qkview サポート情報の取得（参考情報）
================================================

Qkviewとは、BIG-IPの構成情報やログなどの診断情報を取得するためのプログラムで、トラブル時にF5 Supportで一次解析をするために必要となるファイルです。
取得したファイルをサポート受付時に提供いただいたり、`iHealth <https://ihealth.f5.com/>`_ というサポートサイト上にアップロードすることでデコードされた情報を確認することができます。
